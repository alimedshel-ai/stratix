# 📝 سجل التغييرات — Stratix Changelog

## [v4.0] — 2026-02-24

### 🛡️ الأمان (Day 1)
- **إضافة `middleware/security.js`** — طبقة أمان جديدة:
  - `inputSanitizer` — تنظيف المدخلات من XSS (body, query, params)
  - `suspiciousPatternDetector` — كشف أنماط SQL Injection والسكربتات
  - `securityHeaders` — إضافة HSTS, DNS Prefetch Control, Download Options
  - `securityLogger` — تسجيل محاولات الوصول غير المصرح (401/403) والطلبات البطيئة
  - `apiKeyGuard` — حماية إضافية بمفتاح API في البيئة الإنتاجية
- **تعزيز كلمات المرور** — كلمات مرور قوية لجميع الحسابات التجريبية
- **تحديث Super Admin** — كلمة مرور أقوى + هاش bcrypt 10 rounds
- **تكامل في `server.js`** — جميع طبقات الأمان مفعلة قبل الـ routes
