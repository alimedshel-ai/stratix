## 🎬 السيناريو 17: م. سعد (المستثمر/المحلل المالي)

> يدرس الشركات قبل الاستثمار، يحتاج بيانات استراتيجية موثوقة

### المسار:
1. **طلب الوصول** ← بموافقة الشركة المستهدفة
2. **التحقق** ← صلاحية 30 يوم، بيانات 3 سنوات
3. **لوحة المستثمر** ← ملخص استثماري + أداء تاريخي + مخاطر
4. **التحليل العميق** ← مقارنة قطاعية + تحليل منافسة
5. **الاجتماع** ← إعداد أسئلة استثمارية + جدولة مع الإدارة
6. **القرار** ← تقرير استثماري نهائي + توصية + شروط

### لوحة المستثمر تعرض:
- القيمة التقديرية للشركة
- مؤشر النضج الاستراتيجي
- مخاطر الاستثمار
- الأداء عبر 3 سنوات (نمو، ربحية، عملاء، ابتكار)
- مقارنة مع المنافسين والمتوسط القطاعي
- التقارير المصدقة (من المدقق)
- نموذج تقييم مالي (Excel)

### صلاحيات:
✅ يمكن: رؤية البيانات المصدقة، مقارنة قطاعية، تحميل نماذج مالية
❌ لا يمكن: تعديل البيانات، رؤية بيانات غير مصدقة

### الميزات:
| العنصر | التفاصيل |
|--------|----------|
| **الرؤية** | استثمارية — قيمة وتوقعات |
| **البيانات** | تاريخية + مصدقة + قابلة للمقارنة |
| **التحليل** | قطاعي + تنافسي + مالي |
| **القرار** | توصية واضحة + شروط حماية |
| **الوقت** | محدود (30 يوم) — مركّز |

### الفرق عن المدقق:
| | 💰 مستثمر | 🔍 مدقق |
|:---|:---|:---|
| الهدف | قرار استثماري | تصديق بيانات |
| النظرة | للمستقبل (3-5 سنوات) | للماضي (فترة محددة) |
| التحليل | قيمة + مخاطر + فرص | دقة + اكتمال |
| النتيجة | عرض استثماري | تقرير اعتماد |

## 🎬 السيناريو 18: ليلى (الموظفة المنقولة بين الأقسام)

> كانت في التسويق، انتقلت للمبيعات، تحتاج نقل بياناتها وتاريخها

### المسار:
1. **إشعار النقل** ← بريد من HR مع تفاصيل القسم الجديد
2. **تأكيد النقل** ← شاشة توضح: ما ينتقل معها ✅ وما يبدأ جديد 🆕
3. **أول يوم** ← لوحة جديدة مع فريق جديد وأهداف جديدة
4. **الربط الذكي** ← النظام يقترح استغلال علاقاتها القديمة
5. **التقييم المزدوج** ← تقدير التاريخ + تطوير الجديد
6. **التكامل** ← الجمع بين مهارات القسمين = قيمة فريدة

### ما ينتقل معها:
✅ السجل التدريبي، المهارات الموثقة، الإنجازات الكمية
❌ أهداف القسم القديم، KPIs التسويقية

### ما يبدأ جديد:
- أهداف مبيعات المنطقة الغربية
- KPIs جديدة: إيرادات، عملاء، رضا
- فريق جديد + مدير جديد

### الميزات:
| العنصر | التفاصيل |
|--------|----------|
| **السجل** | ينتقل — لا يبدأ من صفر |
| **الذكاء** | يربط القديم بالجديد (علاقات سابقة = فرص) |
| **التقييم** | مزدوج: القديم يُقدر، الجديد يُطور |
| **النمو** | تكامل المهارات — قيمة مضافة فريدة |

### الفرق عن الموظف الجديد:
| | 🔄 منقولة | 🚀 جديد |
|:---|:---|:---|
| البداية | بتاريخ وإنجازات | من الصفر |
| الشبكة | علاقات سابقة | يبني من جديد |
| السرعة | أسرع في الإنجاز | أبطأ في التعلم |

## 🎬 السيناريو 19: د. عبدالرحمن (المتقاعد/الخبير الاستشاري)

> CEO سابق، 35 سنة خبرة، يريد نقل خبرته دون التزام وظيفي

### المسار:
1. **الاكتشاف** ← experts.stratix.com/retired
2. **التقديم** ← نموذج + فيديو 10 دقائق تجريبي
3. **القبول** ← ورشة "كيف تكون مستشاراً رقمياً" + دليل أخلاقيات
4. **أول جلسة** ← استشارة 30 دقيقة، 300 ريال/جلسة
5. **التراكم** ← 25 جلسة، 7,500 ريال، 20 CEO استفادوا
6. **الإرث** ← كتاب إلكتروني تلقائي من أهم مقولاته + بودكاست

### ما يميزه:
- جدول مرن — 10 ساعات/أسبوع
- استشارات قصيرة — 30 دقيقة
- أجر مجزٍ — لكل جلسة
- تراث مهني — كتاب، بودكاست، محفوظات

### الأدوات:
- جدولة جلسات مع CEOs
- ملخص تلقائي لكل جلسة
- تجميع أهم المقولات والنصائح
- إنشاء كتاب إلكتروني تلقائي
- إصدار شهادة "خبير متميز"

### الميزات:
| العنصر | التفاصيل |
|--------|----------|
| **المرونة** | يختار وقته — لا التزام وظيفي |
| **القيمة** | ينقل خبرة 35 سنة — لا يحتفظ بها |
| **الأجر** | 300 ريال/جلسة — يقدّر خبرته |
| **التراث** | كتاب + بودكاست — يبقى |
| **الكرامة** | خبير معتمد — لا "متقاعد فقط" |

### الفرق عن المستشار:
| | 🎖️ متقاعد | 💼 مستشار |
|:---|:---|:---|
| الخبرة | 35 سنة — عميقة | 10 سنوات — متخصصة |
| الشكل | جلسات قصيرة (30 دقيقة) | علاقات طويلة (شهور) |
| الوقت | مرن (10 ساعات/أسبوع) | ملتزم (40 ساعة/أسبوع) |
| التراث | كتاب + بودكاست | علاقات مستمرة |

## 🎬 السيناريو 20: خالد (المُفكِّر في التأسيس)

> يفكر في تأسيس شركة، يحتاج دراسة الفكرة والجدوى قبل المخاطرة

### المسار:
1. **الفكرة** ← يصف فكرته → AI يحلل السوق والمنافسة والفرصة
2. **نموذج العمل** ← Canvas محسّن (القيمة، العملاء، الإيرادات، التكاليف)
3. **الدراسة المالية** ← تقدير إيرادات 3 سنوات + تكاليف + نقطة التعادل + ROI
4. **القرار** ← توصية ذكية: بدء / تحسين / بحث عن شريك

### أدوات جديدة مقترحة:

#### أداة "دراسة الجدوى":
- تقدير الإيرادات (3 سنوات)
- تقدير التكاليف الثابتة والمتغيرة
- نقطة التعادل (Break-even)
- العائد على الاستثمار (ROI)
- الاحتياج التمويلي
- ربط تلقائي مع بيانات Canvas

#### خيارات التكامل المالي:
| الخيار | الوصف | الوقت |
|:---|:---|:---:|
| **أ: تكامل بسيط** | تصدير لـ Excel | 3 أيام |
| **ب: أداة جديدة** | دراسة جدوى مبسطة في المنصة | أسبوع |
| **ج: شراكة** | ربط مع منصة فينتك | شهر |

### خيارات القرار بعد الدراسة:
- [بدء الشركة] ← ينتقل لمسار "ناشئ" (سيناريو 1)
- [تحسين الفكرة] ← يُعيد الدراسة
- [البحث عن شريك] ← يربطه بمستثمر (سيناريو 17)

### الميزات:
| العنصر | التفاصيل |
|--------|----------|
| **التحليل** | AI يحلل السوق والمنافسة فوراً |
| **النموذج** | Canvas محسّن مع بيانات مالية |
| **الجدوى** | أرقام واضحة قبل المخاطرة |
| **القرار** | توصية ذكية مبنية على البيانات |
| **الربط** | يتصل بمسارات أخرى (ناشئ / مستثمر) |
