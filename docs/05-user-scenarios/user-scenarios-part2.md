4. **محاولة الدخول** ← شاشة "حسابك معلق" مع 4 خيارات

### الخيارات عند التعليق:
- ترقية فورية
- تحدث مع الدعم
- تصدير البيانات (PDF/Excel)
- حذف الحساب نهائياً

## 🎬 السيناريو 13: منى (مدخلة البيانات)

> موظفة إدارية، تدخل KPIs يومياً، لا ترى الخطة الكاملة

### المسار:
1. **الدعوة** ← بريد من المدير بصلاحية "إدخال بيانات"
2. **أول دخول** ← بيئة مبسطة جداً — شاشة واحدة
3. **الإدخال اليومي** ← جدول KPIs مع حقول فارغة
4. **التحقق الذكي** ← تنبيه عند أرقام غير منطقية
5. **التأكيد** ← ملخص يومي مع نسبة الجودة
6. **التقرير** ← ملخص أسبوعي تلقائي

### صلاحيات محدودة:
✅ يمكن: إدخال أرقام، تعديل إدخالاتها، رفع Excel
❌ لا يمكن: رؤية الخطة، الأهداف الكبرى، تحليلات المدير

### الميزات:
| العنصر | التفاصيل |
|--------|----------|
| **الصلاحية** | إدخال فقط — لا رؤية ولا تحليل |
| **البساطة** | شاشة واحدة — لا تنقلات |
| **السرعة** | 15 دقيقة/يوم |
| **الجودة** | تحقق ذكي — أخطاء أقل |
| **التتبع** | كل إدخال مسجل — مساءلة |

## 🎬 السيناريو 14: سلطان (المراقب — عضو مجلس الإدارة)

> يريد متابعة أداء الشركة بدون التدخل في التفاصيل

### المسار:
1. **الدعوة** ← بريد من CEO بصلاحية "مراقبة"
2. **أول دخول** ← لوحة مجلس الإدارة (عالية المستوى)
3. **المتابعة** ← مؤشر الصحة + الأهداف الكبرى + المخاطر
4. **التقرير الربعي** ← PDF تلقائي مع ملخص تنفيذي
5. **التفاعل** ← تعليقات فقط — لا تعديل

### لوحة مجلس الإدارة تعرض:
- مؤشر الصحة الاستراتيجية (نسبة مئوية)
- الأهداف الكبرى مع نسب التقدم
- المخاطر الرئيسية
- التقرير الربعي (PDF)

### صلاحيات:
✅ يمكن: رؤية المؤشرات الكبرى، التقارير، التعليق
❌ لا يمكن: تفاصيل المهام، بيانات الموظفين، التعديل

### الفرق عن المدير:
| | 👁️ مراقب | 🎯 مدير |
|:---|:---|:---|
| الرؤية | أهداف كبرى | KPIs تفصيلية |
| التدخل | تعليقات فقط | قرارات + تعديلات |
| الوقت | 10 دقائق/أسبوع | ساعات يومياً |

## 🎬 السيناريو 15: د. نورة (المدققة/المراجعة الخارجية)

> تدقق بيانات الشركة لصالح جهة رقابية أو استثمارية

### المسار:
1. **طلب الدخول** ← رابط مؤقت (7 أيام) مع نطاق محدد
2. **التحقق من الهوية** ← رقم ترخيص + جهة التدقيق
3. **لوحة التدقيق** ← أهداف + KPIs + مصادر البيانات + وثائق مرفقة
4. **التدقيق التفصيلي** ← يرى كل بيان مع مصدره ومن أدخله
5. **الاستفسار** ← يرسل أسئلة للشركة ويتلقى ردوداً موثقة
6. **التقرير النهائي** ← معتمدة / معتمدة مع ملاحظات / غير معتمدة + توقيع إلكتروني

### صلاحيات:
✅ يمكن: رؤية تفصيلية + مصادر البيانات + إضافة ملاحظات + إرسال استفسارات
❌ لا يمكن: تعديل أي بيان — كل تصرف مسجل

### لوحة التدقيق تعرض:
- الأهداف الاستراتيجية مع النسب (هدف vs فعلي)
- مؤشرات الأداء التفصيلية مع مصادر البيانات
- الوثائق المرفقة (تقارير مالية، إثباتات، شهادات)
- حقل ملاحظات التدقيق
- نتيجة التدقيق (معتمدة / مع ملاحظات / غير معتمدة)

### الميزات:
| العنصر | التفاصيل |
|--------|----------|
| **الدخول** | رابط مؤقت — صلاحية محدودة (7 أيام) |
| **النطاق** | فترة محددة + بيانات محددة |
| **الصلاحية** | رؤية فقط + ملاحظات — لا تعديل |
| **الشفافية** | كل تصرف مسجل ومُبلَّغ |
| **التوثيق** | مصادر كل بيان مرفقة |
| **النتيجة** | تقرير رسمي معتمد بتوقيع إلكتروني |

### الفرق عن المراقب:
| | 🔍 مدقق | 👁️ مراقب |
|:---|:---|:---|
| العلاقة | خارجية مستقلة | داخلية (مجلس إدارة) |
| النطاق | فترة محددة | مستمر |
| الصلاحية | تفصيلية + مصادر | عامة فقط |
| القرار | تقرير معتمد | تعليق فقط |
| الوقت | مكثف لمدة محدودة | متابعة دورية |

## 🎬 السيناريو 16: م. عبدالله (المدرب المعتمد)

> يستخدم المنصة لتدريب فرق الشركات — مدرب منهجي لا مستشار

### شروط القبول:
- 5+ سنوات تدريب
- شهادة في الإدارة الاستراتيجية
- اجتياز اختبار المنصة

### المسار:
1. **التقديم** ← طلب + إثبات المؤهلات
2. **القبول** ← رقم شهادة: STX-TR-2024-089
3. **التدريب** ← برنامج 3 أيام (أدوات، إدارة فصول، أعمال)
4. **أول فصل** ← إنشاء فصل تدريبي مع رابط للمتدربين
5. **يوم التدريب** ← متابعة مباشرة لتقدم المتدربين
6. **التقييم** ← تقرير تلقائي + شهادات إتمام
7. **النمو** ← مدرب → متميز → خبير تدريب

### أدوات المدرب:
- إنشاء فصول تدريبية مع رابط مخصص
- متابعة تقدم المتدربين بشكل مباشر (Live)
- تمارين عملية تفاعلية داخل المنصة
- إصدار شهادات إتمام تلقائياً
- تقارير أداء لكل فصل

### الدخل:
| المصدر | المثال |
|--------|--------|
| أجر التدريب | 8,000 ريال/فصل |
| عمولة اشتراكات | 2,500 ريال |
| مكافأة متميز | +10% إضافية |

### الميزات:
| العنصر | التفاصيل |
|--------|----------|
| **الدور** | يُدرّس المنهجية — لا يستشار |
| **الشكل** | فصول منظمة (3 أيام) — لا علاقات طويلة |
| **النتيجة** | متدربون مؤهلون — لا خطط جاهزة |
| **الأدوات** | فصول تفاعلية + متابعة مباشرة |
| **الدخل** | أجر + عمولة + مكافآت |
| **النمو** | مدرب → متميز → خبير تدريب |

### الفرق عن المستشار:
| | 🎓 مدرب | 💼 مستشار |
|:---|:---|:---|
| العلاقة | فصل محدد (3 أيام) | علاقة طويلة (شهور) |
| الهدف | تعليم المنهجية | بناء الخطة للعميل |
| النتيجة | المتدرب يبني بنفسه | المستشار يبني معه |
| العدد | 25 في فصل واحد | 5-20 عميل فردي |
| الأجر | لكل فصل | لكل عميل شهرياً |
