**منطق التوليد في JS:**
```javascript
function generateSW(answers, rules) {
  const strengths = [], weaknesses = [];
  for (const [qId, answer] of Object.entries(answers)) {
    const rule = rules[qId];
    if (!rule) continue;
    if (rule.strength.includes(answer)) strengths.push(rule.strengthLabel);
    else if (rule.weakness.includes(answer)) weaknesses.push(rule.weaknessLabel);
  }
  return { strengths, weaknesses };
}
```

**الإضافة:** زر "ولّد من التشخيص" + 3 مصادر

```
المصدر 1: التشخيص (Phase 0-5)
  → درجة محور ≥ 70% = قوة ("أداء قوي في [المحور]")
  → درجة محور ≤ 40% = ضعف ("فجوة حرجة في [المحور]")
  → مخاطر الامتثال = تهديدات
  → القطاع + الحجم = فرص

المصدر 2: إجابات الإدارات (Phase 6A — dept-deep)
  → strengthRules → نقاط قوة تفصيلية
  → weaknessRules → نقاط ضعف تفصيلية
  → التحديات المختارة = تهديدات إضافية

المصدر 3: الهوية المؤسسية (Phase 7 — لاحقاً)
  → الرؤية = فلتر الأولويات
  → الميزة التنافسية = نقطة قوة محورية
```

**الأقسام:**
```
┌──────────────────┬──────────────────┐
│ 💪 قوة (تلقائي)  │ ⚠️ ضعف (تلقائي) │

├──────────────────┼──────────────────┤
│ 🚀 فرص (تلقائي)  │ 🔴 تهديدات       │
│ + إضافة يدوية    │ + إضافة يدوية    │
└──────────────────┴──────────────────┘

  - أكبر فرصة تُقرّبك: [اختيار]
```
