**منطق التوليد في JS:**
```javascript
function generateSW(answers, rules) {
  const strengths = [], weaknesses = [];
  for (const [qId, answer] of Object.entries(answers)) {
    const rule = rules[qId];
    if (!rule) continue;
    if (rule.strength.includes(answer)) strengths.push(rule.strengthLabel);
    else if (rule.weakness.includes(answer)) weaknesses.push(rule.weaknessLabel);
  }
  return { strengths, weaknesses };
}
```

## Phase 7 — بناء الهوية المؤسسية


## Phase 8 — SWOT المخصص (3 مصادر)

### الملف: تعديل `swot.html`
### الإضافة: زر "ولّد من التشخيص" + قسم ربط بالرؤية

### 3 مصادر للتوليد التلقائي:

```
المصدر 1: التشخيص (Phase 0-5)
  → درجات المحاور → قوة/ضعف موضوعي
  → مخاطر الامتثال → تهديدات
  → القطاع + الحجم → فرص

المصدر 2: إجابات الإدارات (Phase 6)
  → إجابات إيجابية = نقاط قوة تفصيلية
  → إجابات سلبية = نقاط ضعف تفصيلية
  → التحديات المختارة = تهديدات

المصدر 3: الهوية المؤسسية (Phase 7)
  → الرؤية = فلتر الأولويات
  → الميزة التنافسية = نقطة قوة محورية
  → القيم = معايير التقييم
```

### أقسام SWOT المطوّر:

```
القسم 1: SWOT تلقائي (مولّد من البيانات)
  ┌──────────────────┬──────────────────┐
  │ 💪 نقاط القوة    │ ⚠️ نقاط الضعف   │
  │ من الأقسام 1+2+3 │ من الأقسام 1+2+3 │
  ├──────────────────┼──────────────────┤
  │ 🚀 الفرص         │ 🔴 التهديدات     │
  │ من القطاع+الحجم  │ من المخاطر       │
  └──────────────────┴──────────────────┘

القسم 2: إضافات يدوية
  - المستخدم يضيف نقاط إضافية لكل ربع
  - [+ أضف قوة] [+ أضف ضعف] [+ أضف فرصة] [+ أضف تهديد]

القسم 3: الربط بالرؤية ← جديد
  رؤيتك: "[ما كتبه]"
  - أكبر قوة تخدم رؤيتك: [اختيار]
  - أكبر ضعف يعيق رؤيتك: [اختيار]
  - أكبر فرصة تُقرّبك: [اختيار]
  - أكبر تهديد يُبعدك: [اختيار]
```

### الحجم: ~200 سطر إضافية على swot.html
