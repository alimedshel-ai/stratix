## 📊 خريطة الربط — ماذا يقرأ ماذا؟

```
                    يكتب في localStorage        يقرأ من localStorage
                    ────────────────────        ────────────────────

select-type     →  stratix_category
pain-screen     →  stratix_pain                 ← stratix_category
                   stratix_redirected_from

financial-input →  stratix_analysis             ← stratix_pain
                   stratix_financial_v33         ← stratix_category
                   stratix_raw_inputs
                   stratix_financial_size

financial-result → (يقرأ فقط)                   ← stratix_analysis
                                                ← stratix_financial_v33
                                                ← stratix_raw_inputs

financial-report → (يقرأ فقط)                   ← stratix_analysis

action-plan     → (يقرأ فقط)                    ← stratix_analysis (جزئي)

strategy-hub    → (يقرأ فقط)                    ← stratix_analysis
                                                ← stratix_pain

diagnostic-center → stratix_diagnostic_*         ← (مستقل تماماً)

dept-deep       → (يحفظ محلياً فقط)              ← (❌ لا يقرأ شيء من بقية النظام)

scenarios       → (مستقل)                        ← (❌ لا يقرأ شيء)

compliance-analyzer → (مستقل)                    ← (❌ لا يقرأ شيء)

dashboard       → (يقرأ من API)                  ← (❌ لا يقرأ localStorage)
```

**الخلاصة: فقط المسار المالي مربوط داخلياً (input → result → report). كل شي ثاني = جزيرة منفصلة.**
