# 🏗️ خريطة Stratix الكاملة — التصور النهائي

> **التاريخ:** 2026-02-23  
> **الحالة:** وثيقة مرجعية رسمية — المصدر الأساسي لأي تطوير مستقبلي

---

## 👥 أنواع المستخدمين (4 أنواع)

| النوع | الوصف | حجم الشركة | المستوى |
|-------|-------|------------|---------|
| 🧑‍💼 **مستكشف** | شركة ناشئة، أول مرة | 1-20 موظف | مبتدئ |
| 👔 **مدير شركة** | يدير شركة متوسطة/كبيرة | 20-200+ | متوسط/متقدم |
| 📊 **مدير إدارة** | يدير قسماً محدداً | - | متخصص |
| 🧠 **مستشار** | يدير عدة شركات لعملاء | متعدد | خبير |

## 🧭 الهيكل العام — 6 أقسام

```
⚡ العمل اليومي (3)
   ├── أدخل البيانات
   ├── لوحة القيادة
   └── إجراءاتي

🧭 رحلتي الاستراتيجية (6 مراحل)
   ├── 1️⃣ الألم والطموح
   ├── 2️⃣ بنيتنا (DNA + كيانات + إعدادات)
   ├── 3️⃣ تشخيصي (9 أدوات)
   ├── 4️⃣ خياراتي وخطتي (TOWS + أهداف + KPIs + OKRs)
   ├── 5️⃣ تنفيذي (مبادرات + مشاريع + مهام)
   └── 6️⃣ متابعتي (مراجعات + ذكاء + تصحيحات)

📊 التحليل الاستراتيجي (9 أدوات)
   ├── SWOT → PESTEL → بورتر → سلسلة القيمة
   ├── VRIO → القدرات الجوهرية
   └── رحلة العميل → تحليل الفجوات → المقارنة المعيارية

🎯 التخطيط والتنفيذ (4)
   ├── الأهداف (BSC/OKRs)
   ├── مؤشرات الأداء (KPIs)
   ├── المبادرات والمشاريع
   └── الإجراءات التصحيحية

🧰 أدواتي المتقدمة (5)
   ├── استيراد البيانات
   ├── العرض التقديمي الذكي
   ├── سجل النشاطات
   ├── DNA المنظمة
   └── دليل الأدوات

⚙️ الإعدادات (4)
   ├── ملف الكيان
   ├── المستخدمون
   ├── التكاملات
   └── إعدادات عامة
```

## 📊 جدول الصلاحيات الشامل

| القسم | مستكشف | مدير شركة | مدير إدارة | مستشار |
|-------|---------|-----------|------------|---------|
| ⚡ العمل اليومي | ✅ كامل | ✅ كامل | ✅ كامل | ✅ لكل شركة |
| 🧭 الرحلة الاستراتيجية | مرحلة 1 فقط | كل المراحل | ❌ مخفي | ✅ لكل شركة |
| 📊 التحليل الاستراتيجي | SWOT فقط | كل الأدوات (9) | ❌ مخفي | ✅ لكل شركة |
| 🎯 التخطيط | أهداف أساسية | كل شيء | أهداف إدارته | ✅ لكل شركة |
| 🧰 أدوات متقدمة | ❌ مخفي | ✅ كامل | جزئي | ✅ + مقارنة |
| ⚙️ الإعدادات | أساسي | كامل | ملفه فقط | كامل + عملاء |

## 🔗 الخيط الذهبي — تدفق البيانات

```
[DNA + ملف الكيان] → يحدد الهوية
        ↓
[الألم والطموح] → يحدد الوجهة
        ↓
[التحليل الاستراتيجي] → يفهم الواقع
        ↓
[الخيارات الاستراتيجية] → يقرر الطريق
        ↓
[التخطيط] → يترجم إلى أهداف
        ↓
[التنفيذ] → يعمل يومياً
        ↓
[المتابعة] → يراقب ويصحح
        ↓
[التقارير] → يوثق ويتعلم
        ↑
[المستشار الذكي] ← يربط الكل بذكاء
```

## 🤖 المستشار الذكي — يتكيف مع كل مستخدم

| المستخدم | سلوك المستشار |
|----------|---------------|
| مستكشف | "أنصحك تركز على الألم والطموح أولاً" |
| مدير شركة | "لاحظت أن تحليل SWOT ناقص، هل تريد المساعدة؟" |
| مدير إدارة | "مؤشر المبيعات منخفض 15%، هل تريد إنشاء إجراء؟" |
| مستشار | "شركتك 'أ' متأخرة — شركتك 'ب' متميزة — هل تريد نقل الخبرة؟" |

## 🔄 مسار النمو الطبيعي

```
مستكشف (ناشئة) → مدير شركة (متوسطة) → مدير شركة (كبيرة) + مديري إدارات → مستشار
```

- نفس الحساب يبقى، الصلاحيات تتوسع
- البيانات القديمة تستمر
- الأدوات تنفتح تلقائياً مع النمو
