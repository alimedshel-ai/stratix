# 🗺️ تصور مسار تجربة العميل — User Journey Plan

> التاريخ: 21 فبراير 2026
> الحالة: **ينتظر الاعتماد**

---

## 🎯 المشكلة

العميل يدخل المنصة ويجد **60 صفحة** بدون توجيه واضح:
- ما يعرف يبدأ من وين
- الـ Sidebar فيه كل الروابط مرة واحدة
- الـ Dashboard ما يوجّه للخطوة التالية
- صفحات متداخلة (analysis.html vs tool-detail.html)

---

## ✅ الحل: مسار مرحلي واضح (5 مراحل)

```
المرحلة 1        المرحلة 2        المرحلة 3        المرحلة 4        المرحلة 5
التأسيس    →    التشخيص    →    التخطيط    →    التنفيذ    →    المتابعة
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🏗️              🔍              🎯              🚀              📊
```

---

## 📋 تفصيل كل مرحلة

### المرحلة 1: التأسيس 🏗️
> "من أنت؟ وماذا تريد أن تحقق؟"

| الخطوة | الصفحة | الشرط للإكمال |
|--------|--------|--------------|
| 1.1 إنشاء حساب | `/signup` | تسجيل ناجح |
| 1.2 إعداد المنظمة | `/onboarding` | اختيار الشركة + القطاع + الكيان |
| 1.3 تحديد الرؤية والرسالة | `/settings` | كتابة الرؤية والرسالة والقيم |

**النتيجة:** كيان جاهز مع نسخة استراتيجية أولى (DRAFT)

---

### المرحلة 2: التشخيص 🔍
> "أين أنت الآن؟"

| الخطوة | الصفحة | الشرط للإكمال |
|--------|--------|--------------|
| 2.1 تحليل SWOT | `/tool?code=SWOT` | إضافة عنصر واحد على الأقل في كل قسم |
| 2.2 تحليل PESTEL | `/tool?code=PESTEL` | إكمال التحليل |
| 2.3 تحليل Porter | `/tool?code=PORTER` | إكمال التحليل |
| 2.4 تقييم شامل *(اختياري)* | `/assessments` | إكمال تقييم واحد |

**النتيجة:** صورة واضحة عن الوضع الحالي

---

### المرحلة 3: التخطيط 🎯
> "إلى أين تريد أن تصل؟"

| الخطوة | الصفحة | الشرط للإكمال |
|--------|--------|--------------|
| 3.1 التوجهات الاستراتيجية | `/directions` | إضافة توجه واحد على الأقل |
| 3.2 الأهداف الاستراتيجية | `/objectives` | إضافة 3 أهداف على الأقل |
| 3.3 مؤشرات الأداء | `/kpis` | ربط KPI واحد على الأقل بكل هدف |
| 3.4 الخيارات والمخاطر | `/choices` | تحديد خيار استراتيجي واحد |

**النتيجة:** خطة استراتيجية واضحة بأهداف قابلة للقياس

---

### المرحلة 4: التنفيذ 🚀
> "كيف ستنفذ؟"

| الخطوة | الصفحة | الشرط للإكمال |
|--------|--------|--------------|
| 4.1 المبادرات | `/initiatives` | إنشاء مبادرة واحدة مرتبطة بهدف |
| 4.2 المشاريع | `/projects` | إنشاء مشروع واحد |
| 4.3 المهام | `/tasks` | إنشاء وإكمال مهمة واحدة |
| 4.4 إدخال بيانات KPI | `/kpi-entries` | إدخال قيمة فعلية واحدة |

**النتيجة:** بدء التنفيذ الفعلي

---

### المرحلة 5: المتابعة 📊
> "هل أنت في المسار الصحيح؟"

| الخطوة | الصفحة | الشرط للإكمال |
|--------|--------|--------------|
| 5.1 مراجعة دورية | `/reviews` | إكمال مراجعة واحدة |
| 5.2 لوحة الذكاء | `/intelligence` | مشاهدة التنبيهات |
| 5.3 التقارير | `/auto-reports` | توليد تقرير واحد |
| 5.4 التصحيحات | `/corrections` | إنشاء إجراء تصحيحي (إن وُجد) |

**النتيجة:** دورة تحسين مستمر

## 🖥️ التأثير على الواجهة

### 1. الـ Sidebar (التغيير الأهم)

**قبل (الحالي):**
```
├── الرئيسية
├── الكيانات
├── القطاعات
├── المستخدمين
├── التقييمات
├── الأهداف
├── المؤشرات
├── ... (30+ رابط مرة واحدة)
```

**بعد (المقترح):**
```
📍 أنت في: المرحلة 2 — التشخيص
━━━━━━━━━━━━━━━━━━━━━━━━━

🏠 الرئيسية

📌 المسار الاستراتيجي:
  ✅ 1. التأسيس
  🔵 2. التشخيص        ← الحالية
  │   ├── SWOT ✅
  │   ├── PESTEL ⏳     ← التالية
  │   └── Porter 🔒
  ⬜ 3. التخطيط
  🔒 4. التنفيذ
  🔒 5. المتابعة

⸻⸻⸻⸻⸻⸻⸻⸻
⚙️ الإعدادات
👤 حسابي
```

**القاعدة:** المراحل المقفلة (🔒) تظهر لكن ما تنضغط حتى تكمل المرحلة السابقة.

---

### 2. الـ Dashboard

**قبل:** إحصائيات عامة بدون توجيه

**بعد:** بطاقة بارزة في الأعلى:

```
┌─────────────────────────────────────────────────┐
│  📍 أنت في المرحلة 2: التشخيص                   │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━                   │
│  ✅ SWOT مكتمل   ⏳ PESTEL (60%)   🔒 Porter    │
│                                                  │
│  [▓▓▓▓▓▓▓▓░░░░░░░] 40% مكتمل                   │
│                                                  │
│  ← الخطوة التالية: أكمل تحليل PESTEL            │
│  [ابدأ الآن →]                                   │
└─────────────────────────────────────────────────┘
```

### 3. الصفحات المتقدمة

الصفحات التالية **لا تظهر في المسار الأساسي** لكن متاحة من قائمة "أدوات متقدمة":

| الصفحة | السبب |
|--------|-------|
| `/ceo-dashboard` | لوحة تنفيذية خاصة |
| `/admin-panel` | إدارة المنصة |
| `/benchmarking` | مقارنات متقدمة |
| `/simulation-lab` | محاكاة سيناريوهات |
| `/org-dna` | تحليل DNA المنظمة |
| `/okrs` | إطار OKRs بديل |
| `/risk-map` | خريطة المخاطر |
| `/live-board` | لوحة عرض حية |
| `/ai-center` | مركز الذكاء الاصطناعي |
| `/ai-presentation` | عروض AI |
| `/stakeholders` | أصحاب المصلحة |
| `/strategic-calendar` | التقويم الاستراتيجي |
